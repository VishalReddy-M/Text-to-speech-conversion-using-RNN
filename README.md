# Text-to-speech-conversion-using-RNN


## 📌 Overview

This project implements a **Text-to-Speech (TTS) Conversion System** using a **Recurrent Neural Network (RNN)**. The model learns sequential patterns in text and generates speech representations that can be converted into audio output.

The project demonstrates the application of deep learning techniques in Natural Language Processing (NLP) and Speech Processing.

---

## 🚀 Features

- Convert text into speech using a Recurrent Neural Network (RNN)
- Text preprocessing and tokenization
- Sequence modeling for speech generation
- Model training and evaluation
- Simple and easy-to-understand implementation
- Built using Python and TensorFlow/Keras


## 🧠 Model Architecture

The project uses a Recurrent Neural Network (RNN) for sequential text processing.

**Workflow**

```
Input Text
      │
      ▼
Text Cleaning
      │
      ▼
Tokenization
      │
      ▼
Word Embedding
      │
      ▼
Recurrent Neural Network
      │
      ▼
Dense Layer
      │
      ▼
Speech Representation
      │
      ▼
Generated Audio
```
